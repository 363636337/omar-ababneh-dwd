# omar-ababneh-dwd
DARPA-X Airship Electrical Architecture
Overview:

This repository contains the system-level electrical architecture and wiring diagram developed for the DARPA-X Airship project during my internship. The diagram illustrates the integration of avionics, propulsion, telemetry, navigation, and power-distribution subsystems required for autonomous airship operation.

Project Objectives:

Develop a complete electrical integration architecture for the airship platform.
Define power-distribution pathways between batteries, voltage regulators, avionics, and propulsion systems.
Document communication interfaces between flight-control and peripheral devices.
Support system assembly, troubleshooting, and future development efforts.
